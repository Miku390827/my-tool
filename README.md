# My Tool

日々の作業を効率化するためのシンプルで強力なツール。

## 概要

My Tool は、作業をもっとスマートにするための軽量ツールです。高速・安全・カスタマイズ性を重視して設計されています。

## 機能

- **高速** — 軽量設計で、どんな環境でもサクサク動作
- **安全** — データはローカルで処理し、プライバシーを最優先
- **カスタマイズ** — 柔軟な設定でワークフローに合わせて最適化

## 使い方

`index.html` をブラウザで開くだけで動作します。

```bash
open index.html
```

## 開発

```bash
git clone https://github.com/Miku390827/my-tool.git
cd my-tool
```
